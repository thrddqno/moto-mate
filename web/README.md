# Moto Mate Web

React + Vite Progressive Web App for Moto Mate.

## Commands

- `npm run dev` starts the local Vite server.
- `npm run build` runs TypeScript and creates the production PWA build.
- `npm run preview` serves the production build locally.
- `npm run lint` checks the frontend source.
- `npm test` runs Vitest.

## Environment

Copy `.env.example` to `.env.local` and fill in Firebase Web SDK values plus the backend URL.

The default API URL is `http://localhost:8081/api/v1`.
